Delete Operation for Two Strings

Key Observation

We can only delete characters from both strings.

The characters that should not be deleted are the ones belonging to the Longest Common Subsequence (LCS).

Intuition

  • The LCS represents the maximum set of characters common to both strings that can be kept.
  • All remaining characters must be deleted to make both strings equal.

Therefore,

Deletions from String 1 = n - LCS

Deletions from String 2 = m - LCS

Total deletions:

text id="dgo0ws"
(n - LCS) + (m - LCS)

= n + m - 2 × LCS

Steps

text id="u63zlh"
String 1
     │
String 2
     │
Find LCS
     │
Delete all characters not in LCS
     │
Answer = (n - LCS) + (m - LCS)

Complexity

  • LCS: O(N × M)
  • Time: O(N × M)
  • Space: O(N × M) (can be optimized to O(M))

Recognition Pattern

If the question asks:

  • Delete Operation
  • Make Two Strings Equal
  • Only Deletions Allowed

Immediately think:

Find LCS
      ↓
Keep LCS
      ↓
Delete remaining characters

Answer = (n - LCS) + (m - LCS)
