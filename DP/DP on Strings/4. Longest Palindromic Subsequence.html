Longest Palindromic Subsequence (LPS)

Key Observation

A palindrome reads the same forward and backward.

So,

text
Longest Palindromic Subsequence (LPS)
          =
LCS(original string, reversed string)
⚙️ Approach
  1. Reverse the original string.
  2. Find the Longest Common Subsequence (LCS) between:
  • Original string
  • Reversed string
  1. The LCS length is the Longest Palindromic Subsequence.

Why does it work?

A palindromic subsequence appears in the same order in the original string and also in its reversed string.

Therefore, finding the Longest Common Subsequence between the original and reversed strings gives the Longest Palindromic Subsequence.

Steps

text
Original String
       │
Reverse the String
       │
Find LCS(original, reversed)
       │
Answer = LPS Length

Complexity

  • Reverse String: O(N)
  • LCS: O(N²)
  • Time: O(N²)
  • Space: O(N²) (can be optimized to O(N))

Recognition Pattern

If the question asks for:

  • Longest Palindromic Subsequence

Immediately think:

text
Reverse the string
        +
Apply LCS
🟢
Important

✅ Longest Palindromic Subsequence → Solve using LCS(original, reverse).

❌ Longest Palindromic Substring → Cannot be solved using LCS because substrings must be contiguous.
